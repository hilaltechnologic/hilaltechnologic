# Yuk , Kenalan dengan Astro: Framework Super Cepat untuk Bangun Website Modern

Di tengah hiruk-pikuk dunia frontend yang makin rame kayak pasar malam, muncul satu nama yang mencuri perhatian: **Astro**. Framework satu ini datang bukan sekadar ikut tren, tapi benar-benar nyodorin solusi nyata buat developer yang pengen situs cepet, SEO-friendly, dan hemat JavaScript.

> "Ship less JavaScript." — Astro Team

Dan bro... dia bukan cuma omdo.

---

## 🌌 Apa Itu Astro?

Astro adalah framework web **static site generator (SSG)** dan **server-first** yang dirancang untuk membangun website secepat kilat. Cocok buat blog, portofolio, dokumentasi, landing page, sampai ecommerce ringan.

### Fitur Unggulan Astro:

- 🚫 Zero JS by default
- 💧 Partial hydration
- 🔀 Mendukung React, Vue, Svelte, Solid
- 🛠️ Build statis & SSR
- ✍️ Markdown & MDX friendly
- 🎨 Tailwind CSS ready

---

## ⚙️ Kenapa Harus Pakai Astro?

### 1. Performa Gila Cepet
Astro nyajiin halaman HTML statis tanpa overkill JavaScript. Skor Lighthouse bisa mentok 100 kayak main cheat.

### 2. Framework-Agnostic
Lo bisa gabungin komponen dari berbagai framework.

### 3. Dokumentasi Rapi
Belajar Astro tuh gak bikin kepala panas.

### 4. Cocok Buat Blog
Support Markdown, MDX, dan SEO built-in. Mantap kan?
